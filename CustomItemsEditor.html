Custom Items Editor

The Custom Items Editor can be accessed via the Edit menu.

The Custom Items Editor will then pop up. Note you will not be able to click outside of the popup window while it's open.

At the bottom of the window are the buttons for interacting with the editor.

You can add a new item with the ADD button

A window will pop up asking you to enter in the new item's description. This is the description as it will appear in the document.

Note: you cannot give two items the same description, and custom items cannot have the same description as the existing default items. Costs Drawer will prevent you from doing so and show an error message.

The new item will appear in the list as shown below.

When an item is in the list, you can then select the outlay and professional values for editing. Simply double click on the area, or navigate using the keyboard arrow keys and press enter. (Note this also works for editing the description if you wish to change it.)

Editing the outlay and professional values works the same way as entering the description.

You can see the item outlay has now been changed.

You can add as many items as you need.

To delete an item, simply make sure you have the correct item highlighted, either using the mouse or the keyboard, then click on DELETE.

You will be asked to confirm the deletion. Make sure to double check the description is for the correct item.

Items created in the Custom Items Editor will be automatically added to the bottom of the Items list in the right column. The order of the items shown in editor is the order the items will appear in the list. You can move items up or down this list if you want to make one easier to get to. Simply select the item you want to move...

Note that if you hover over an item, you can see the full description as a tooltip.

And press either of the MOVE buttons (arrows indicate the direction).

Now the item with the description "This is a second item." has been moved up the list.

When a change has been made to the list, the SAVE button will appear. You must click on this button to save your changes to the list.

Closing the editor is done via the CLOSE button. If you have made changes and haven't saved yet, you will be asked to confirm that you wish to close without saving your changes.

When you close the editor, scroll to the bottom of the Items list in the right column area to see them. Double-clicking on one will insert it into the document at the current position, just as with any other items.

FINAL NOTE: Circuit Court and High Court have separate items lists, so if you want an item to be available in both, you must add it to both manually. The Custom Items List you are editing depends on the jurisdicition of the bill you are currently editing.
