What's New in Costs Drawer 2.3.2

See release notes for v2.3.2

This release fixes many small bugs and issues:

  • Temporarily disabled ability to highlight text as it had some odd behaviour.
  • Fixed a bug that could prevent instructions fee items from being edited.
  • Changed "split tool" so that you can now split on an individual character rather than splitting by line.
  • Fixed layout issues with Plaintiffs and Defendant positions on the front page.

  • Fixed some slowdown when editing item descriptions.
  • Opening page now has a download link to new versions.
  • Create Instructions Fee Item is now clickable as soon as General Instructions Fee is in view.

What's New in Costs Drawer 2.3.0

See release notes for v2.3.0

This release adds a new feature to the front page so users can add text below the Record number and above the first Plaintiff name. You can now change the "Costs of behalf of" between Plaintiff and Defendant (further features are planned for this). It also introduces the ability to highlight text, which then allows you to cut or copy the highlighted text. Note this feature is still somewhat in development, so might not behave exactly as expected.

Importantly this version also fixes a number of bugs:

Most Notable Bug Fixes:

  • A bug that could cause Costs Drawer to crash when you attempted to edit an Instructions Fee item.
  • A recently introduced bug that caused exported Circuit Court bills to not include a fee for Drawing Costs and Copy.

Details:

  • Fixed a recently introduced bug where instructions fee items could crash Costs Drawer when clicking on them.
  • Fixed an issue where Circuit Court costs would not include fee for Drawing Costs and Copy when exported to Excel format.
  • Added 3 rows to the front page below the record number where users can add info about their Circuit region or whatever other info might be needed.
  • Fixed an issue with the dictionary that caused words beginning with the letter 'P' to be always marked as incorrect.
  • Added ability to highlight text, along with ability to copy, cut and delete the highlighted text. (still in development)
  • Fixed an issue introduced by a recent change where, when editing an item's description, the text would not appear to fit the column correctly when zoomed in or out.
  • Fixed: "Move Item Up" grayed-out high-resolution sprite was using the wrong sprite.
  • Move Items Up/Down buttons now appear gray when the selected item can't be moved.
  • Fixed issues with the File Menu when in large text / high resolution mode.
  • Tooltips should now always appear above other objects and text.
  • Fixed the Search box appearing over "Help" menu when in high resolution mode.
  • Added a button to change the document between Plaintiff and Defendant (further features in relation to this are planned)
  • Removed the print feature. Users should instead export to Excel and then print from there.
  • "Circuit Court" and "High Court" text is now bold when exported to Excel format.
  • Record Number no longer remains selected after confirming (pressing enter).
  • Pasted text will now include hyphens (i.e. " - ").

What's New in Costs Drawer 2.2

See release notes for Costs Drawer 2.2.2

See release notes for Costs Drawer 2.2.1

Features

  • Costs Drawer will now check for updates and download them as they come out (user is prompted before download)
  • Spellcheck dictionary now contains well over 60,000 words! (up from 8500) Costs Drawer will also keep any words you've added yourself separately.
  • Added ability to right-click on a word and add it to the dictionary.
  • Added links to YouTube video tutorials and "What's New" page

Changes

  • Costs Drawer will no longer show a popup message warning users when offline. Instead, a notice is displayed at the bottom of the tips area.
  • Tips area can no longer be disabled.
  • Highlighting a button now changes the cursor to a hand pointer
  • Highlighting color changed
  • Changed "Toggle Small Screen Mode" text in View menu to "Toggle Large Menu Buttons", for clarity.
  • Split tool mode can be exited using "ESC" key

Fixes

  • Fixed an issue where Excel would print Sheet Name and Page Numbers on every page of printed bill.
  • Costs Drawer will now more elegantly scale to smaller resolutions.
  • Fixed a graphical bug that affected the whole screen making it blurry.
  • Fixed a graphical bug where toolbar buttons would 'wobble' when scrolling through the document.
  • Fixed some user interface issues
  • Improved optimization of spell checker, should be much faster now.
  • Fixed odd behavior in [blue text] tags.
  • Fixed the spell checker marking acceptable contractions and punctuation as incorrect. NOTE: contractions should still be avoided in Bills of Costs regardless!
  • Fixed a graphical error on the opening screen that occurred when turning TIPS on and off.

Other

  • Made some changes to clean up the File menu.
  • Many optimizations and bug fixes, as well as other under-the-hood improvements.
